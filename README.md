# Contact Form

## Description
A responsive contact form built with HTML, CSS, and JavaScript featuring real-time validation and toast notifications.

## Features
- Input validation for all fields
- Red/green border feedback
- Custom error messages
- Toast notification on success
- Form resets after successful submission

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)

## How to Use
1. Fill in all the fields
2. Click Send Message
3. A success notification will appear if all fields are valid

## What I Learned
Building this project helped me strengthen my understanding of how HTML, CSS, and JavaScript work together. I learned how to manipulate the DOM by grabbing elements and dynamically changing their styles and content. I also learned how to validate form inputs using regular expressions, return values from functions, and use event listeners to handle form submission. One of my favourite parts was building the toast notification by combining JavaScript's classList and setTimeout with CSS positioning and transitions.
